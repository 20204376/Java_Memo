=> 자바 프로그램의 경우

-> jvm이 운영체제와 소통하여 환경에 맞게 설정해줌 -> 운영체제에 의존적이지 않음(이식성 높다)

자바의 기본 구조
1) jvm(java virtual machien)
: 운영체제에 맞춰서 java로 만든 프로그램을 실행시켜 줌
javac : .java(사람의 언어로 된 파일) 컴파일
java : .class(컴퓨터의 언어로 된 파일) 실행

2)JRE(JAVA RUNTIME ENVIRONEMNT)
:JVM을 실행한다
JVM을 실행할 때 필요한 파일 등을 보유

3) JDK(JAVA DEVELOPMENT KIT)
:JRE을 포함한다
JRE외에 개발에 필요한 도구 보유

4)IDE(통합개발환경)
:개발의 편의성을 높이기 위해서 사용하는 도구

-이클립스, VISUALSTUDIOCODE, 인텔리제이 ...
-이클립스= 무료, 다양한 단축기 제공하여 개발 편의성 UP

window > Preferences > compiler 검색 >  JDK Compliance에서 level 변경
(기본 18 -> 17로 변경) > Apply

5) 이클립스 우측 상단의 창문 모양 클릭 > JAVA 선택 > Open
