변수
어떤 값을 저장하기 위한 저장공간

1) 변수를 만드는 방법(변수의 선언)
데이터(값)의 형태와 변수의 이름(변수명) 2가지 정보가 필요하다

자료형(데이터의 형태) 변수명; ;=마침표

ex) int(자료형) a(변수명); : a라는 공간에는 정수의 값만 들어올 수 있다

2)변수의 값을 저장하는 방법
	변수명 = 값;
	(=) 대입연산자 : 오른쪽 값을 왼쪽에 저장
	EX) A = 10;

int a;
a = 10;

3)선언과 동시에 값을 저장
자료형(데이터의 형태) 변수명 = 값;
ex) int a = 10;

4) 변수에 저장된 값 사용
변수명을 사용한다

ex) syso(a);

5) 여러개의 변수를 선언하는 방법
자료형 변수명1, 변수명2,...

ex) int a,b,c;
	a=10;
	b=11;

6) 변수명 규칙
-띄어쓰기 불가능
-스네이크 표기법 (_),(int_myAge) 
-카멜표기법(소문자로 시자그 단어가 바뀔때 대문자로 시작) 
-파스칼 표기법(클래스명을 만들 때 사용, 클래스명 앞글자는 대문자)
-특수문자 사용 불가능(_제외)
-숫자로 시작 불가능 but 중간, 끝은 삽입 가능
-키워드(예약어) 사용 불가능하다
	키워드(예약어) : 자바에서 문법적으로 사용되고 있는 단어
	ex for, if, class,...
- ★의미있는★ 변수명을 되도록 사용하자
ex)name, age (클래스명과 구별하기 위해 소문자로 시작하자)
