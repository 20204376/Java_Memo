반복문(for문)
for(최식;조건식;증감식){

{

초기식 : 처음에 설정할 값
조건식 : 조건이 참이면 영역안의 문장 실행시키고 거짓이면 for문 탈출
증감식 : 반복할 때마다 값을 얼마나 증감할지 정하는 식

반복문(while문)
while(조건식) {
	조건식이 참일동안 실행하는 문장;
}
while문과 for문의 차이
for문 : 몇 번 반복할지 알 때
while문: 몇번 반복할지 모를 때

반복문(do~while문)

do{
 반복실행할 문장;
}while(조건식);

while문과 동일하지만 최소 한 번은 무조건 실행해야할 때 사용한다.
반복실행할 문장 먼저 실행하고 그 다음에 조건식 판별

기타제어문
반복문에서 사용한다. 단break문은 switch문에서도 사용가능
break : 즉시 해당 중괄호 영역을 탈출한다.
	-if문은 적용되지 않음(if문 안에서 사용시 if문을 감싸고 있는 중괄호 탈출)
	-아래 코드 실행시키지 않음

continue : 즉시 다음 반복문으로 넘어간다
	-아래 코드를 실행시키지 않음
